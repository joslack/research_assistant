# research_assistant
An application utilizing LangChain and the OpenAI API to index pdfs and act as a teacher or a search engine.

# project structure

project/  
│  
├── data/  
│   ├── pdfs/  
│   └── indexed_data/  
│  
├── src/  
│   ├── preprocess.py  
│   ├── index.py  
│   ├── search.py  
│   ├── teach.py (optional)  
│   └── main.py  
│  
├── ui/ (optional)  
│   ├── templates/  
│   └── static/  
│  
├── requirements.txt  
└── README.md
